# Code Style

## Java conventions
- Имена классов: CamelCase (RecommendationsService)
- Имена методов: camelCase (calculateScore)
- Константы: UPPER_SNAKE_CASE

## Структура пакетов

├── controller
├── service
├── repository
├── model
└── config

## Code Review правила
- Максимальная длина метода: 30 строк
- Обязательные комментарии к сложной логике
- Тесты для нового функционала
